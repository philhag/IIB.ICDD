IIB.ICDD.Model Namespaces
Contains all models to represent an ICDD container according to ISO 29157
Namespaces
Namespace Description
IIB.ICDD.Model Contains all models to represent an ICDD container according to ISO 29157
IIB.ICDD.Model.Container Contains all models derived from the Container ontology
IIB.ICDD.Model.Container.Document Contains all models for representing documents
IIB.ICDD.Model.Container.ExtendedDocument Contains all models for representing documents according to the ExtendedDocument ontology https://w3id.org/exdoc
IIB.ICDD.Model.DynamicSemantics Contains all models derived from the DynamicSemantics ontology (only included in the draft of the ISO standard)
IIB.ICDD.Model.DynamicSemantics.Properties Contains all models derived from the DynamicSemantics ontology for representing properties
IIB.ICDD.Model.DynamicSemantics.Relations Contains all models derived from the DynamicSemantics ontology for representing relations
IIB.ICDD.Model.ExtendedLinkset Contains all models derived from the ExtendedLinkset ontology from Part 2 of the standard
IIB.ICDD.Model.Interfaces Contains interfaces for handling the model
IIB.ICDD.Model.Linkset Contains all models derived from the Linkset ontology
IIB.ICDD.Model.Linkset.Identifier Contains all models derived from the Linkset ontology for representing identifiers
IIB.ICDD.Model.Linkset.Link Contains all models derived from the Linkset ontology for representing links
IIB.ICDD.Model.PayloadTriples Contains models for handling the payload triples in the container
